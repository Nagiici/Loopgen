A verified runner for AI coding loops

Run your AI's coding loop — and prove its work actually passed.

Any model can write an AGENTS.md. loopgen does the part a stateless model can't.

It runs a bounded loop, executes your real test / lint / build and gates success on the exit codes, enforces forbidden-path and iteration limits, and writes a tamper-evident, hash-chained audit log + proof report. Works with Claude Code, Codex, Cursor, and local models.

$ npx loopgen run test-repair GitHub Docs

Run a loop, verify it, and leave proof it passed — or fail on a broken test / a forbidden secret.

Any model can generate a config. loopgen does what a stateless model can't: execute, verify, prove, and enforce.

01
Prove the work

Run the loop and prove it passed

After you (or any agent — Claude Code, Cursor, Codex) make a bounded change, loopgen run diffs the working tree, runs your real verification commands, gates pass/fail on the exit codes, checks whether forbidden paths were touched, and writes a hash-chained audit + a human-readable proof report. Exits 0/1, so it drops straight into CI or a git hook.

# run + verify a loop; exits 0 on pass, 1 on fail
npx loopgen run test-repair
02
Drive & enforce

Let loopgen drive a local model — and block bad writes

Driven mode goes from detection to prevention: loopgen drives a local model (Ollama / OpenAI-compatible) through a bounded loop and validates every proposed action before it lands — forbidden-path writes are blocked, non-allowlisted commands never run, and iterations/timeouts are hard-capped. Local-first: only your model is called; API keys by env-var name only.

npx loopgen run test-repair --mode driven \
  --adapter ollama --ollama-model qwen2.5-coder
03
Govern

Turn the ledgers into team evidence — and a CI gate

Every run appends to a per-repo, hash-chained ledger. loopgen audit rolls many devs'/repos' ledgers into a report + a self-contained HTML dashboard (no server), and audit check is a merge gate — blocking PRs that lack passing, untampered proof. There's a ready-made GitHub Action.

npx loopgen audit aggregate ./repos --html gov.html
npx loopgen audit check --require test-repair --require-no-violations --require-chain
